# Security Policy

Awesome Operations Research is a curated resource list, not an executable software package. Security issues are therefore most likely to involve unsafe, malicious, misleading, or compromised links.

## Reporting Unsafe Links

Please report:

- Malicious or suspicious links.
- Links that redirect to unsafe content.
- Resources that appear to impersonate official projects.
- Download pages that bundle unexpected software.
- Broken links that now point to unrelated or unsafe content.

Open a [broken link issue](.github/ISSUE_TEMPLATE/broken_link.md) or contact the maintainers privately if public disclosure could put users at risk.

## What to Include

- The file and section where the link appears.
- The resource name and URL.
- What behavior looked unsafe or suspicious.
- A safer official replacement link, if known.

## Maintainer Response

Maintainers should remove or replace unsafe links as soon as practical and may temporarily remove a resource while verifying it.
