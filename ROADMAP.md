# Roadmap

## Near-Term

- Verify external links.
- Add missing descriptions.
- Improve tags.
- Add more official tutorials.
- Invite community contributions.

## Medium-Term

- Add more industry case studies.
- Add more international OR societies.
- Add more teaching resources.
- Add curated beginner projects.

## Long-Term

- Submit to major awesome-list indexes.
- Build a companion website if useful.
- Add automated quality checks.
- Maintain annual refresh cycles.

## Maintenance Cadence

- Automated link checks run weekly; results are reviewed at least monthly or quarterly.
- A major resource refresh (descriptions, new tools, retired projects) is done annually.
- Broken links are removed or replaced as they are found or reported.
- Inactive software is retained only when it is historically important, and labeled as such.
- Community pull requests are reviewed for relevance, quality, and official sourcing before merging.
