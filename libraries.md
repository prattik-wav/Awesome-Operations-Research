## Python

- [CasADi](https://web.casadi.org/) - Framework for algorithmic differentiation and nonlinear optimization.
- [cylp](https://github.com/coin-or/CyLP) - Python interface to COIN-OR CLP and CBC.
- [CVXPY](https://www.cvxpy.org/) - Python modeling language for convex optimization.
- [DEAP](https://deap.readthedocs.io/) - Evolutionary algorithms framework for genetic algorithms, genetic programming, and other metaheuristics.
- [docplex](https://ibmdecisionoptimization.github.io/docplex-doc/) - IBM Decision Optimization modeling package for Python.
- [gurobipy](https://www.gurobi.com/documentation/current/refman/py_python_api_overview.html) - Official Python API for Gurobi Optimizer.
- [highspy](https://github.com/ERGO-Code/HiGHS/tree/master/highspy) - Python interface to the HiGHS optimization solver.
- [mip](https://www.python-mip.com/) - Python tools for modeling and solving mixed-integer linear programs.
- [NetworkX](https://networkx.org/) - Python package for graph algorithms and network analysis.
- [OR-Tools](https://developers.google.com/optimization) - Python APIs for CP-SAT, routing, scheduling, flows, assignment, and linear solving.
- [PuLP](https://coin-or.github.io/pulp/) - Python modeling package for linear and integer programming.
- [Pyomo](https://www.pyomo.org/) - Python algebraic modeling language for optimization.
- [PySCIPOpt](https://github.com/scipopt/PySCIPOpt) - Python interface to SCIP.
- [python-constraint](https://github.com/python-constraint/python-constraint) - Python package for finite-domain constraint satisfaction problems.
- [pymoo](https://pymoo.org/) - Python framework for single-objective and multi-objective optimization, including evolutionary algorithms.
- [SciPy Optimize](https://docs.scipy.org/doc/scipy/reference/optimize.html) - Optimization module in SciPy for nonlinear, constrained, least-squares, and root-finding problems.
- [SimPy](https://simpy.readthedocs.io/) - Process-based discrete-event simulation framework.

## Julia

- [BlackBoxOptim.jl](https://github.com/robertfeldt/BlackBoxOptim.jl) - Julia package for derivative-free and black-box optimization.
- [Convex.jl](https://jump.dev/Convex.jl/stable/) - Julia package for disciplined convex optimization.
- [Graphs.jl](https://juliagraphs.org/Graphs.jl/stable/) - Julia library for graph data structures and algorithms.
- [JuMP](https://jump.dev/) - Julia modeling language for mathematical optimization.
- [MathOptInterface](https://jump.dev/MathOptInterface.jl/stable/) - Solver abstraction layer used by JuMP and related packages.
- [Optim.jl](https://julianlsolvers.github.io/Optim.jl/stable/) - Julia package for continuous optimization.
- [Plasmo.jl](https://github.com/plasmo-dev/Plasmo.jl) - Julia package for graph-structured optimization models.
- [PowerModels.jl](https://lanl-ansi.github.io/PowerModels.jl/stable/) - Julia package for power network optimization models.
- [SDDP.jl](https://sddp.dev/) - Julia package for stochastic dual dynamic programming.
- [StochasticPrograms.jl](https://github.com/martinbiel/StochasticPrograms.jl) - Julia framework for stochastic programming.

## R

- [CVXR](https://cvxr.rbind.io/) - R package for disciplined convex optimization.
- [igraph](https://r.igraph.org/) - R interface to igraph for graph analytics and network modeling.
- [lpSolve](https://cran.r-project.org/package=lpSolve) - R interface to lp_solve for linear and integer programming.
- [ompr](https://dirkschumacher.github.io/ompr/) - Algebraic modeling package for mixed-integer linear programming in R.
- [optimx](https://cran.r-project.org/package=optimx) - Extended optimization tools for R.
- [ROI](https://roi.r-forge.r-project.org/) - R Optimization Infrastructure for solver-independent optimization.
- [rPref](https://cran.r-project.org/package=rPref) - Preference query package useful for Pareto and multi-criteria analysis.
- [Rsymphony](https://cran.r-project.org/package=Rsymphony) - R interface to the SYMPHONY mixed-integer programming solver.
- [simmer](https://r-simmer.org/) - Discrete-event simulation package for R.

## MATLAB

- [CVX](http://cvxr.com/cvx/) - MATLAB-based modeling system for disciplined convex programming.
- [Global Optimization Toolbox](https://www.mathworks.com/products/global-optimization.html) - MATLAB toolbox for global search, genetic algorithms, particle swarm, and surrogate optimization.
- [MATPOWER](https://matpower.org/) - MATLAB package for power system simulation and optimization.
- [Optimization Toolbox](https://www.mathworks.com/products/optimization.html) - MATLAB toolbox for linear, quadratic, nonlinear, and least-squares optimization.
- [TOMLAB](https://tomopt.com/) - Commercial MATLAB optimization environment with interfaces to multiple solvers.
- [YALMIP](https://yalmip.github.io/) - MATLAB modeling toolbox for optimization and control.

## C and C++

- [Ceres Solver](http://ceres-solver.org/) - C++ library for large-scale nonlinear least-squares and bundle adjustment.
- [COIN-OR libraries](https://www.coin-or.org/projects/) - C and C++ solvers and libraries for LP, MIP, NLP, decomposition, and related OR tasks.
- [GLPK](https://www.gnu.org/software/glpk/) - C library and command-line tools for LP and MIP.
- [HiGHS](https://highs.dev/) - C++ optimization solver for LP, MIP, and QP with C interfaces.
- [Ipopt](https://coin-or.github.io/Ipopt/) - C++ nonlinear programming solver.
- [OR-Tools](https://developers.google.com/optimization) - C++ optimization suite with routing, CP-SAT, and graph algorithms.
- [OSQP](https://osqp.org/) - C implementation of a convex quadratic programming solver.
- [SCIP](https://www.scipopt.org/) - C optimization suite for MIP, MINLP, and constraint integer programming.

## Java

- [Choco Solver](https://choco-solver.org/) - Java constraint programming solver.
- [JGraphT](https://jgrapht.org/) - Java graph theory and graph algorithms library.
- [ojAlgo](https://www.ojalgo.org/) - Java library for optimization, linear algebra, and financial mathematics.
- [OR-Tools](https://developers.google.com/optimization) - Java APIs for Google's optimization suite.
- [Timefold Solver](https://timefold.ai/) - Java/Kotlin planning and scheduling solver for vehicle routing, timetabling, rostering, and related problems.

## JavaScript and TypeScript

- [glpk.js](https://github.com/jvail/glpk.js) - JavaScript build of GLPK for solving LP and MIP models in browser or Node.js.
- [javascript-lp-solver](https://github.com/JWally/jsLPSolver) - JavaScript linear programming solver with integer support.
- [jsLPSolver](https://github.com/JWally/jsLPSolver) - JavaScript LP/MILP solver package used in browser and Node workflows.
- [OR-Tools WebAssembly experiments](https://github.com/google/or-tools) - Official OR-Tools repository; JavaScript usage usually depends on community or WebAssembly builds.

## Go

- [gonum](https://www.gonum.org/) - Numeric, graph, linear algebra, and optimization packages for Go.
- [goptuna](https://github.com/c-bata/goptuna) - Bayesian optimization framework for hyperparameter and black-box optimization.
- [gorgonia](https://gorgonia.org/) - Go library for differentiable computation useful for optimization and machine learning experiments.
- [go-dsp](https://github.com/mjibson/go-dsp) - Signal processing utilities that can support simulation and analytics workflows.
- [go-graph](https://github.com/yourbasic/graph) - Graph algorithms package for Go.
- [go-hep/groot](https://github.com/go-hep/hep) - Scientific data tooling that can support simulation-heavy analysis pipelines.
- [gocron](https://github.com/go-co-op/gocron) - Go job scheduling library useful for operational automation around optimization services.
- [optimize](https://pkg.go.dev/gonum.org/v1/gonum/optimize) - Gonum subpackage for local optimization algorithms.

## Other Languages and Interfaces

- [AMPL APIs](https://ampl.com/api/latest/) - APIs for using AMPL from Python, MATLAB, R, Java, C++, and other languages.
- [GAMS APIs](https://www.gams.com/latest/docs/apis/) - APIs for connecting GAMS models to external languages and systems.
- [MiniZinc Interfaces](https://www.minizinc.org/doc-latest/en/installation.html) - Interfaces and command-line tooling for MiniZinc workflows.
- [OPL and CPLEX APIs](https://www.ibm.com/docs/en/icos) - IBM APIs for modeling and solving optimization problems across supported languages.
