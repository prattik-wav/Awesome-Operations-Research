# Awesome Operations Research

[![Awesome](https://awesome.re/badge.svg)](https://github.com/sindresorhus/awesome)
[![License: CC0-1.0](https://img.shields.io/badge/License-CC0--1.0-lightgrey.svg)](LICENSE)

A curated knowledge base for Operations Research: optimization, simulation, stochastic systems, analytics, decision science, software, datasets, books, courses, papers, careers, teaching, research, and industry practice.

Operations Research (OR) uses mathematical modeling, optimization, simulation, stochastic processes, analytics, and decision science to improve complex systems. This repository is designed as a navigation hub for students, researchers, instructors, practitioners, consultants, and engineers.

## New to Operations Research? Start with These 5 Resources

- Beginner book: [Introduction to Operations Research](books.md#general-operations-research) - Classic broad introduction to OR modeling and methods.
- Free course: [MIT OpenCourseWare optimization and OR courses](courses.md#introductory-operations-research) - Open university-level lecture material.
- Beginner tool: [OpenSolver](softwares.md#spreadsheet-and-teaching-tools) - Spreadsheet-based optimization tool for learning LP and MIP models.
- Modeling library: [PuLP](libraries.md#python) - Beginner-friendly Python library for linear and integer programming.
- Community: [Operations Research Stack Exchange](meetups.md#online-communities) - Q&A forum for OR models, algorithms, solvers, and practice.

## Maintenance

Resources may be removed or marked as deprecated when:

- The maintainer explicitly says the project is no longer maintained.
- The link is broken and no stable replacement exists.
- The project has been inactive for several years and is not historically important.
- The description is promotional, misleading, or no longer accurate.
- The resource has been superseded by an official successor.

Lightweight maintenance scripts are available in [scripts/](scripts/) for internal link checks and resource counts.
